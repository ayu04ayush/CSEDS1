Special Characters

HTML requires certain characters to be escaped. Examples:

  • &amp; renders as &
  • &lt; renders as <
  • &gt; renders as >
  • Other symbols: © (copyright), ™ (trademark), ± (plus/minus), é (&eacute;)

Nested Lists

Nested lists help group sub-items under a parent item:

  • Module 1
    • Lesson 1.1
    • Lesson 1.2
  • Module 2
    • Lesson 2.1
    • Lesson 2.2
      • Exercise 2.2.1
      • Exercise 2.2.2

Mixed Lists (Ordered + Unordered)

Mixing list types is common for structured curriculums:

  1. Week 1
    • Reading: Chapter 1
    • Assignment: Problem set 1
  2. Week 2
    • Reading: Chapter 2
    • Lab: Setup development environment
